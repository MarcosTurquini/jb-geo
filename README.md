# Sitio Web Corporativo – Empresa Geológica

Sitio web institucional desarrollado para una empresa del rubro geológico, enfocado en una presentación profesional de sus servicios, identidad corporativa y una correcta visualización en dispositivos móviles y de escritorio.

---

## Descripción

Este proyecto consiste en una **página web estática** diseñada para una empresa geológica, con el objetivo de brindar una presencia digital sólida, clara y profesional.  
El sitio prioriza la **experiencia de usuario**, el **diseño responsive** y una estructura clara de contenidos orientada a clientes y socios comerciales.

---

## Tecnologías utilizadas

- HTML5  
- CSS3  
- JavaScript (ES6+)  

---

## Características principales

- Diseño responsive (mobile-first)
- Estructura clara y profesional
- Secciones informativas sobre servicios geológicos
- Optimización visual para dispositivos móviles
- Código limpio y organizado
- Compatible con los principales navegadores modernos
